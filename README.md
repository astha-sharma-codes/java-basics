# Java Basics
This repository contains Java programs created while learning core programming concepts and problem-solving techniques.

## Topics Covered
- Variables and Data Types
- Conditional Statements (if-else)
- Loops
- Arrays
- Strings
- Pattern Programs
- Number Based Problems
- Searching Techniques

## Programs Included
- Hello World
- Even and Odd Number Check
- Reverse a Number
- Palindrome Number
- String Palindrome
- Fibonacci Series
- Prime Number Check
- Star Pattern (Nested Loops)
- Maximum Element in an Array
- Frequency of an Element in an Array
- Linear Search
- Binary Search
- Count Vowels in a String

## Purpose
To strengthen Java fundamentals and improve logical and problem-solving skills through hands-on practice.
